## INITIALISATION
Modifier le fichier:
* .env

Puis executer les commander suivantes:
* ``composer install``
* ``php/bin console d:d:c``
* ``php/bin console make:migration``
* ``php/bin console doctrine:migrations:migrate``
* ``symfony serve``
