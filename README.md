</p>


<h3>INSTALLATION</h3>
<p>Cloner le repo.</p>
<p>Modifier le fichier:<br>
* .env<br>

Puis executer les commander suivantes:<br>
* ``composer install``<br>
* ``php/bin console d:d:c``<br>
* ``php/bin console make:migration``<br>
* ``php/bin console doctrine:migrations:migrate``<br>
* ``symfony serve``
</p>
